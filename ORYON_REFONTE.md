# Oryon — refonte produit

Cette version recentre le projet sur une interface de streaming plus lisible :

- Accueil centré sur les lives natifs Oryon.
- Onglet Twitch séparé pour la recherche, l’embed et les lives suivis.
- Comptes utilisateurs Oryon.
- Connexion Twitch conservée.
- Prototype WebRTC natif avec affichage public du salon dès le lancement.
